# 🎓 College Information Website

A responsive College Information Website developed using **Flask, SQLAlchemy, SQLite, HTML, CSS, and JavaScript**. The website provides students with information about the institution, courses, faculty, gallery, and admissions, while offering a secure admin dashboard for content management.

## 🌐 Live Demo

**Website:** https://pssshe.onrender.com

**Admin Login:** https://pssshe.onrender.com/admin/login

---

# ✨ Features

## Student Portal

- 🏠 Home page with college overview
- ℹ️ About Us section
- 📚 Courses page
- 👨‍🏫 Faculty information
- 🖼️ Gallery
- 📝 Online Admission Form
- 📱 Responsive design

- 🔐 Secure Admin Login
- 📊 Dashboard with statistics
- 📝 View Admission Applications
- 🖼️ Upload & Manage Gallery Images

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/college-information-website.git
```

## Go to Project Folder

```bash
cd college-information-website
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Application

```bash
python app.py
```

The application will start at:

```
http://127.0.0.1:5000
```
